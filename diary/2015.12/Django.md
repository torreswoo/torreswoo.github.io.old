# Django

### Django 설치

```python
$> pip install django  // 모듈설치
$> django-admin.py startproject {project-name}  // 프로젝트설치

$> cd {project-name}
$> django-admin.py startapp {app-name}  // app설치
( /Users/1002475/.pyenv/versions/my-virtual-env-2.7.8/lib/python2.7/site-packages/django/bin/django-admin.py startproject proj_django )

$> python manage.py runserver // 실행
```
